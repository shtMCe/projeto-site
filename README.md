# Projeto HTML e CSS - Universidade Federal de Viçosa

## Descrição

Este projeto foi desenvolvido com o objetivo de praticar os conceitos básicos de HTML e CSS através da criação de uma página informativa sobre a Universidade Federal de Viçosa (UFV), com foco no campus de Rio Paranaíba.

A página apresenta informações sobre a localização do campus e alguns dos cursos oferecidos pela instituição.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts

---

## Funcionalidades

* Exibição do nome da universidade.
* Apresentação do campus de Rio Paranaíba.
* Exibição de imagem ilustrativa do campus.
* Lista com alguns cursos oferecidos pela instituição.
* Estilização utilizando CSS externo.

## Conceitos praticados

Durante o desenvolvimento deste projeto foram utilizados os seguintes conceitos:

### HTML

* Estrutura básica de um documento HTML.
* Uso das tags:

  * `header`
  * `article`
  * `h1`
  * `h2`
  * `p`
  * `img`
  * `ul`
  * `li`

### CSS

* Seletores básicos.
* Estilização de textos.
* Manipulação de espaçamentos com `margin` e `padding`.
* Uso de `border` e `border-radius`.
* Aplicação de fontes externas utilizando Google Fonts.
* Organização do código utilizando arquivo CSS separado.
